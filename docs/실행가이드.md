# LetsStudySaaS 실행 가이드

이 문서는 LetsStudySaaS 프로젝트의 백엔드(Backend)와 프론트엔드(Frontend)를 로컬 환경에서 실행하는 방법을 안내합니다.

---

## 1. 백엔드 (Backend) 실행 방법

백엔드는 Python과 Flask를 기반으로 작성되어 있습니다.

### 요구 사항
- Python 3.x 버전을 설치해주세요.

### 실행 단계

1. **프로젝트 폴더로 이동**
   터미널을 열고 백엔드 디렉토리로 이동합니다.
   ```bash
   cd backend
   ```

2. **가상 환경(Virtual Environment) 활성화**
   이미 `venv` 폴더가 생성되어 있으므로 가상 환경을 활성화합니다.
   - **Windows (Command Prompt / PowerShell):**
     ```bash
     .\venv\Scripts\activate
     ```
   - **macOS / Linux:**
     ```bash
     source venv/bin/activate
     ```

3. **필요한 패키지 설치 (최초 1회)**
   `requirements.txt`가 있다면 아래 명령어로 의존성을 설치합니다.
   ```bash
   pip install -r requirements.txt
   ```
   *(참고: `run.py` 실행 시 `flask`, `flask_restx`, `python-dotenv` 등이 필요합니다.)*

4. **서버 실행**
   아래 명령어를 통해 서버를 실행합니다.
   ```bash
   python run.py
   ```

5. **결과 확인**
   기본적으로 `http://localhost:5000` (또는 `http://127.0.0.0:5000`) 주소로 백엔드 서버가 실행되며, 브라우저를 통해 접속 시 API 응답을 확인할 수 있습니다. API 문서는 Swagger 형태로 제공될 수 있습니다.

---

## 2. 프론트엔드 (Frontend) 실행 방법

프론트엔드는 React와 Vite를 기반으로 작성되어 있습니다.

### 요구 사항
- Node.js (버전 18 이상 권장) 및 npm이 설치되어 있어야 합니다.

### 실행 단계

1. **프로젝트 폴더로 이동**
   새 터미널 창을 열고 프론트엔드 디렉토리로 이동합니다.
   ```bash
   cd frontend
   ```

2. **의존성(패키지) 패키지 설치**
   최초 실행 시, 프로젝트에 필요한 Node 모듈들을 설치해야 합니다.
   ```bash
   npm install
   ```
   *(참고: 이미 `node_modules` 폴더가 있다면 이 과정을 생략해도 무방합니다.)*

3. **개발 서버 실행**
   아래 명령어를 입력하여 Vite 개발 서버를 시작합니다.
   ```bash
   npm run dev
   ```

4. **결과 확인**
   터미널에 출력되는 로컬 접속 주소 (예: `http://localhost:5173/`)를 `Ctrl + Click`하거나 브라우저에 입력하여 화면이 정상적으로 나타나는지 확인합니다.
